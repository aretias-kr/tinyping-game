# 티니핑 이름 맞추기 게임

## 실행 방법

1) 프로젝트 폴더로 이동
```powershell
cd C:\Users\areti\Documents\tinyping-game
```

2) 로컬 서버 실행
```powershell
python -m http.server
```

3) 브라우저에서 열기
```
http://localhost:8000
```

## 참고
- 파일을 직접 더블클릭해서 열면 `fetch`가 차단될 수 있습니다. 반드시 로컬 서버로 실행해 주세요.
- 이미지/이름 데이터는 `data/mapping.json`에 저장되어 있습니다.
